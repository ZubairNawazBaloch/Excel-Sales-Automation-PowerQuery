# Excel Sales Automation (Power Query & Power Pivot)

## Project Overview
This project automates the process of combining monthly sales files from a dropzone folder. Instead of manually copying and pasting data, the Power Query ETL pipeline automatically extracts, cleans, and consolidates the files into a single model for analysis.

## Key Features
* **Folder Automation:** Built a dynamic connection to a dropzone folder, allowing new sales files to be appended instantly upon refreshing the data.
* **Data Transformation:** Cleansed messy headers, adjusted data types, and handled missing values automatically.
* **Advanced Data Modeling:** Used Power Pivot and DAX to create a high-level performance dashboard tracking regional revenue and product trends.

## Tech Stack Used
* **Microsoft Excel**
* **Power Query** (ETL / Folder combination)
* **Power Pivot & DAX** (`CALCULATE`, `ALL`)
